# Thesis pCLCA Repository

This repository supports a Master's thesis on prospective consequential life cycle assessment (pCLCA) of end-of-life pathways for a mass timber curtain wall case study.

It contains the minimum documentation needed to audit the modelling workflow, including:
- appendix-linked supporting materials
- foreground model documentation
- market and marginal supplier logic
- background-system QA and freeze records
- uncertainty and reproducibility notes

## Repository structure
- `appendices/` thesis appendix support materials
- `docs/` short methodological and reproducibility notes
- `data/` raw, processed, and metadata files
- `scripts/` modelling and post-processing scripts
- `results/` selected tables, figures, and logs
- `archive/` superseded or supporting materials

## Thesis appendix linkage
- Appendix A: Stage D substitution and claim governance
- Appendix B: Foreground models, proxies, parameters, and regionalization
- Appendix C: Background-system construction and QA
- Appendix D: Market boundaries and marginal supplier mixes
- Appendix E: Cross-indicator and supplementary result material
- Appendix F: Uncertainty propagation and sensitivity analysis
- Appendix G: Repository guide and reproducibility support

## Reproducibility note
This repository is intended to provide an auditable record of the thesis workflow. Some large databases, licensed background data, and generated outputs may not be redistributed directly.
