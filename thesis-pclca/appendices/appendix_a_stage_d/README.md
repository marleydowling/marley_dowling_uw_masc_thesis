# Appendix A Support Files

This folder stores support materials for **Appendix A - Stage D Substitution Governance Pack**.

It is intended to support auditability without duplicating the appendix prose in the thesis.

## What is included
- `table_a1_stage_d_claim_ledger.csv` - machine-readable version of the route-level Stage D claim ledger.
- `table_a2_product_market_mapping.csv` - machine-readable product-to-market mapping table.
- `table_a3_governance_variant_register.csv` - machine-readable governance-variant register.
- `node_proxy_crosswalk.csv` - implementation crosswalk linking thesis route logic to builder node names and scenario-suffixed code patterns.
- `claim_basis_notes.md` - short implementation notes clarifying how Appendix A maps to the final builder architecture.
- `validation_log.txt` - run log showing which expected builder files were found and how many rows were written.
- `MANIFEST.csv` - appendix-folder inventory and file-purpose register.

## What is intentionally excluded
- Full duplicated appendix prose.
- Brightway databases or licensed background inventories.
- Historical / superseded builder versions not needed to support the final thesis narrative.

## Interpretation rule
The thesis appendix remains the authoritative interpretive text. The files here act as supporting evidence and implementation traceability only.
